HƯỚNG DẪN ĐẶT QR CODE THANH TOÁN
==================================

1. Đặt file ảnh QR code của bạn vào thư mục này với tên: qr_code.png

2. File ảnh nên có:
   - Định dạng: PNG hoặc JPG
   - Kích thước: Tối thiểu 300x300px (khuyến nghị 500x500px)
   - Chất lượng: Rõ nét, dễ quét

3. QR code nên chứa thông tin:
   - Số tài khoản: 7621983825
   - Tên chủ tài khoản: CHENH HUNG THO
   - Ngân hàng: BIDV - CN PHU MY PGD CHAU DUC

4. Sau khi đặt file, QR code sẽ tự động hiển thị trong popup thanh toán.

Lưu ý: Nếu file không tồn tại, hệ thống sẽ hiển thị thông báo lỗi.
